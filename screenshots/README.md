# Screenshots

This folder contains visual documentation of the ImpactGuru Mini CRM application.

## How to Add Screenshots

Save the following screenshots in this folder:

1. **01-login.png** - Login page screenshot (127.0.0.1:8000/login)
2. **02-admin-dashboard.png** - Admin dashboard view (127.0.0.1:8000/dashboard)
3. **03-customers-list.png** - Customer management page (127.0.0.1:8000/customers)
4. **04-orders-list.png** - Order management page (127.0.0.1:8000/orders)
5. **05-users-management.png** - User management page (127.0.0.1:8000/users)

Simply save each screenshot image with the corresponding filename above.
